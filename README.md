# Tic-Tac-Toe
by Lawrence Taylor


A simple browser-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features

- Two-player gameplay
- Custom player names
- Win detection (rows, columns, diagonals)
- Tie detection
- Game restart functionality
- Clean, simple interface

## How to Play

1. Enter player names (optional)
2. Click "Start New Game" to begin
3. Players take turns clicking empty squares
4. First player to get three in a row wins
5. If all squares fill with no winner, it's a tie
6. Click "Restart Game" to play again

## Files

- `index.html` - Game structure
- `style.css` - Simple styling
- `script.js` - Game logic
